🐶 Chicken Woof Dog Biscuits Website

 Description
Chicken Woof Dog Biscuits is a simple dog food ordering website developed using HTML, CSS, and JavaScript. The website allows customers to place orders by entering their name, phone number, delivery location, and quantity of dog biscuits. The current delivery date is automatically displayed.

Features
- Dog biscuits product display
- Customer order form
- Quantity selection (grams/kg)
- Phone number input
- Delivery location input
- Automatic current date display
- Simple and user-friendly interface

Technologies Used
- HTML
- CSS
- JavaScript

 How to Run
1. Download the project files.
2. Save the image file in the project folder.
3. Open `index.html` in any web browser.
4. Fill in the order details and click **Place Order**.
